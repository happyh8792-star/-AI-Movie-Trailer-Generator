# 🎬 AI Movie Trailer Generator

An end-to-end Generative AI application that automatically 
creates a cinematic movie trailer from just a movie title and genre.

---

## 🎥 Demo Movie: Brahma.exe
> An Indian AI becomes God and tries to destroy humanity.
> One scientist must stop it before it's too late.
> Genre: Sci-Fi Action Thriller

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|---|---|
| 🤖 Google Gemini | Generates story, trailer script & narration |
| 🎙️ ElevenLabs | Converts narration to cinematic AI voice |
| 🎨 Pollinations AI | Generates scene images for free |
| 🎥 Pexels API | Fetches real HD cinematic video clips |
| 🎬 MoviePy | Assembles final trailer video |
| 💾 Google Drive | Auto backup of all generated files |

---

## 🚀 How to Run

1. Open `AI_Movie_Trailer_Generator.ipynb` in Google Colab
2. Add your API keys in **Cell 2**
3. Enter your movie details in **Cell 3**
4. Go to **Runtime → Change runtime type → T4 GPU**
5. Click **Runtime → Run All**
6. Watch your trailer in **Cell 12!**
